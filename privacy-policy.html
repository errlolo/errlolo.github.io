Política de Privacidad

Nombre de la aplicación: InfoLluvia

Desarrollador: Manuel de Pedro

Esta aplicación no recopila datos personales del usuario. La aplicación utiliza conexión a Internet exclusivamente para consultar información pública sobre el estado de embalses y precipitaciones en España. No se almacenan datos personales. No se comparten datos con terceros. Esta app usa Google AdMob para mostrar publicidad. AdMob podría recoger y procesar datos como ID de publidad, información de dispositivo y uso de datos. Si tiene alguna pregunta puede contactar en: errlolo@gmail.com

Para más información, visite: https://policies.google.com/privacy
